# YZ Exercise - מחולל אימונים חכם 💪

אפליקציית web מתקדמת ליצירת תוכניות אימון מותאמות אישית, עם ממשק מרהיב ותמיכה מלאה במובייל ודסקטופ.

## ✨ תכונות

### 🎯 תכונות ליבה
- **יצירת אימונים מותאמת אישית** - בחירת כמות תרגילים, רמת קושי וציוד זמין
- **אלגוריתם חכם** - מגוון קבוצות שריר באותו אימון
- **ממשק רספונסיבי** - מתאים לאייפון, אנדרואיד, טאבלט ומחשב
- **עיצוב מודרני** - אנימציות חלקות וצבעוניות מרהיבות

### 💾 ניהול אימונים
- **שמירה אוטומטית** - שמירת אימונים ב-LocalStorage
- **היסטוריה** - צפייה ב-10 האימונים האחרונים
- **ייצוא והדפסה** - הדפסת תוכנית האימון
- **שיתוף** - שיתוף האימון דרך WhatsApp, מייל וכו'

### ⏱️ טיימר מובנה
- טיימר אימון עם התחל/עצור/אפס
- מעקב אחר זמן האימון

### 🎨 חוויית משתמש
- **אנימציות מהממות** - כל רכיב מונפש בצורה חלקה
- **פידבק ויזואלי** - הודעות הצלחה/שגיאה אינטראקטיביות
- **לינקים להדרכות** - קישורים ישירים לסרטוני YouTube
- **תמיכה בעברית** - ממשק דו-לשוני מלא

עשוי עם HTML5, CSS3 ו-Vanilla JavaScript טהור - ללא frameworks!
