# YZ Exercise - מחולל אימונים חכם 💪

אפליקציית web מתקדמת ליצירת תוכניות אימון מותאמות אישית, עם ממשק מרהיב ותמיכה מלאה במובייל ודסקטופ.

## ✨ תכונות

### 🎯 תכונות ליבה
- **יצירת אימונים מותאמת אישית** - בחירת כמות תרגילים, רמת קושי וציוד זמין
- **אלגוריתם חכם** - מגוון קבוצות שריר באותו אימון
- **ממשק רספונסיבי** - מתאים לאייפון, אנדרואיד, טאבלט ומחשב
- **עיצוב מודרני** - אנימציות חלקות וצבעוניות מרהיבות

### 💾 ניהול אימונים
- **שמירה אוטומטית** - שמירת אימונים ב-LocalStorage
- **היסטוריה** - צפייה ב-10 האימונים האחרונים
- **ייצוא והדפסה** - הדפסת תוכנית האימון
- **שיתוף** - שיתוף האימון דרך WhatsApp, מייל וכו'

### ⏱️ טיימר מובנה
- טיימר אימון עם התחל/עצור/אפס
- מעקב אחר זמן האימון

### 🎨 חוויית משתמש
- **אנימציות מהממות** - כל רכיב מונפש בצורה חלקה
- **פידבק ויזואלי** - הודעות הצלחה/שגיאה אינטראקטיביות
- **לינקים להדרכות** - קישורים ישירים לסרטוני YouTube
- **תמיכה בעברית** - ממשק דו-לשוני מלא

## 🚀 התקנה

### דרך 1: GitHub Pages (מומלץ)

1. **צור ריפו חדש בגיטהאב:**
   - לך ל-GitHub.com
   - לחץ על "New Repository"
   - תן שם (למשל: `yz-exercise`)
   - לחץ "Create Repository"

2. **העלה את הקבצים:**
```bash
git init
git add .
git commit -m "🚀 Initial commit - YZ Exercise App"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/yz-exercise.git
git push -u origin main
```

3. **הפעל GitHub Pages:**
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main → /root
   - Save

4. האתר יהיה זמין ב:
   `https://YOUR-USERNAME.github.io/yz-exercise/`

## 📊 עדכון התרגילים מ-Excel

הקוד תומך **ישירות** בקובץ JSON שהומר מ-Excel!

### שלבי עדכון:

1. **ערוך את `YZEX.xlsx`** - הוסף/שנה תרגילים באקסל

2. **המר ל-JSON:**
   - לך לאחד מהאתרים:
     - https://beautifytools.com/excel-to-json-converter.php
     - https://tableconvert.com/excel-to-json
     - https://products.aspose.app/cells/conversion/excel-to-json
   - העלה את `YZEX.xlsx`
   - הורד את ה-JSON

3. **שמור בשם `exercises.json`** (תחליף את הקובץ הקיים)

4. **העלה לגיטהאב:**
```bash
git add exercises.json
git commit -m "עדכון תרגילים"
git push
```

**זהו!** האתר יתעדכן אוטומטית תוך דקה ⚡

### מבנה הקובץ הנדרש:

הקוד תומך במבנה שמגיע מהמרת Excel:

```json
{
  "Sheet1": [
    {
      "שם": "שכיבות סמיכה",
      "קבוצת שריר": "חזה",
      "רמת קושי": "קל",
      "סוג ציוד": "משקל גוף",
      "קישור": "https://youtube.com/..."
    }
  ]
}
```

**הערה:** העמודות ב-Excel צריכות להיות:
- `שם` - שם התרגיל
- `קבוצת שריר` - חזה/גב/רגליים/כתפיים/בטן
- `רמת קושי` - קל/בינוני/קשה
- `סוג ציוד` - משקל גוף/דאמבלים/TRX/גומיה
- `קישור` - URL להדרכת וידאו

## 🐛 פתרון בעיות

### התרגילים לא נטענים
1. בדוק ש-`exercises.json` באותה תיקייה כמו `index.html`
2. פתח את ה-Console בדפדפן (F12) וחפש שגיאות
3. ודא שה-JSON תקין - אפשר לבדוק ב: https://jsonlint.com/

### הממשק לא נראה טוב במובייל
- נקה cache של הדפדפן (Ctrl+Shift+Delete)
- רענן את הדף בכוח (Ctrl+F5)

### LocalStorage לא עובד
- אל תשתמש ב-Private/Incognito mode
- ודא שהדפדפן תומך ב-LocalStorage (כל הדפדפנים המודרניים תומכים)

### שגיאה "Unsupported JSON format"
המבנה של ה-JSON לא נכון. ודא שיש:
- `{ "Sheet1": [ ... ] }` - אם המרת מ-Excel
- או מערך פשוט `[ ... ]` - אם יצרת ידנית
